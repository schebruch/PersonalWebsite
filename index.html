Social Media Messaging Application
Fall 2018

During the Fall 2018 semester, my team and I created an application that allows users to post messages to the front-end. Using the Oauth 2.0 authentication API, clients log in with their @lehigh.edu domain accounts. Once logged in, users can vote, comment, edit and delete messages, as well as block users. In addition to posting text, users may also attach images and PDF files. With both a web and android UI, along with a Java backend, the appliction used dependency frameworks including Maven, Gradle, and NPM for efficient dependency management. In addition, use of languages and platforms including HTML, CSS, JavaScript, Bootstrap, and Handlebars was imperative to a clean, user-friendly experience.

Predictive Model For Credit Analysis
Fall 2017

A common problem for banks is that they may end up losing profit if they loan to defaulters. My team and I built various predictive models (such as Decision Trees, Logistic Regression, and Naive Bayes) using the R programming language to determine whether the bank should loan to a particular customer. Models were visualized using ROC Curves and analyzed using the Expected Profit Framework: the best model was the one that yielded the most gain.
